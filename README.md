# archive-the-leyu

## 项目简介

本项目是一个用于归档和发布多个独立 HTML 页面的仓库。仓库中存储的页面内容来自不同来源，以静态 HTML 形式保存，便于长期查阅和分享。

本仓库不针对任何特定域名或网站，仅作为页面资料的存档与展示用途。

## 目录结构

```
archive-the-leyu/
├── pages/           # 存放所有归档的 HTML 页面
├── assets/          # 页面引用的公共资源（图片、样式、脚本等）
└── README.md        # 本文件
```

- `pages/` 目录下每个子文件夹或文件对应一个独立的归档页面。
- `assets/` 目录存储页面所需的公共静态资源。

## 页面归档说明

- 所有页面均为独立 HTML 文件，可直接在浏览器中打开浏览。
- 页面内容保持原始状态，仅做必要的格式调整以确保兼容性。
- 归档时间以文件提交历史为准，不额外标注日期。

## 维护说明

- 本仓库不定期更新，新增页面会以 Pull Request 或直接提交的方式加入。
- 欢迎通过 Issues 提交页面归档建议或报告问题。
- 不接受涉及违规、侵权或敏感内容的归档请求。
- 仓库维护者保留对页面内容进行筛选和移除的权利。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 进入 `pages/` 目录，用浏览器打开任意 HTML 文件即可查看。
3. 也可通过 GitHub Pages 或其他静态托管服务部署后在线访问。

本仓库内容仅供学习与参考，不构成任何形式的授权或承诺。具体页面内容的版权归原作者所有。
